## 🗂️ Projets

### 🤖 FFBB MCP Server
> *Le 1er serveur MCP dédié à la Fédération Française de Basketball.*

Connecte ton assistant IA (Claude, Gemini…) aux données live de la FFBB : scores en direct, calendriers, classements, clubs, salles. **Aucune clé API requise.**

[![Voir le projet](https://img.shields.io/badge/GitHub-FFBB--MCP--Server-181717?style=for-the-badge&logo=github)](https://github.com/nickdesi/FFBB-MCP-Server)

### 🚌 Gerzat Live
> *Transport en commun en temps réel pour les habitants de Gerzat.*

Application open data qui affiche les prochains passages Bus & Train en temps réel. Parce que l'open data, ça doit aussi servir localement.

[![Voir le projet](https://img.shields.io/badge/GitHub-BusTrainGerzat-181717?style=for-the-badge&logo=github)](https://github.com/nickdesi/BusTrainGerzat)
